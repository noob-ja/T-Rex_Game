# GA T-Rex

![](https://github.com/noob-ja/T-Rex_Game/raw/master/screenshot.png)

### Description:
A recreated version of the famous Chrome T-Rex in Python.
Genetic algorithm is implemented.

### Instructions and Prerequisites:   
To run this game:  
* Make sure you have Python installed along with pygame (http://www.pygame.org/) library
* Run main.py

##### Controls:
* Press UP or SPACE to make T-Rex jump over the obstacles
* Press DOWN key to crouch
* Avoid getting hit by obstacles.
